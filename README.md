## 🚀 Overview
This application runs on **Python + Flask**.

Below are the instructions to set up your environment, install dependencies, and launch the application.

---
# Setup Instructions

## 🐍 1. Create & Activate a Virtual Environment

### Create the environment

python3 -m venv venv

### Activate it (macOS / Linux)
source venv/bin/activate

## 📦 2. Install Dependencies

pip install -r requirements.txt


## 🔥 3. Run the Flask Application

make clear
make run

##   4. Accessing the server

 Open on a browser: http://0.0.0.0:5001


##   5. Populate the Database with some data

make seed
